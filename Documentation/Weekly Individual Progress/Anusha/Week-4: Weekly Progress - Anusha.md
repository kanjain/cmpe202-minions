Week-4: Creation story Navigation

Every Game has a background story and a set of instructions/rules to be followed toachieve its goal
In this game, the minions that appear need to be saved by clicking. There needs to be a screen that explains this tothe user

Howdo we navigate to the screen?
Home->Howto->Story

As of Now a place holder screen caled 'Story' has been integrated

Building the prototype
While building the prototype, there is a transition from home to Story screen that is created.
Steps followed:

1. Create a story-screen sprite using tools online like flamingtext.com
2. Create a state called Story 
3. Link a button on the home screen to transition to this state
4. In the state, load the newly created sprite
